# Codex Platform Architecture

## Architecture view
- **Interfaces:** CLI, IDE extension, Codex app, and Codex web/cloud
- **Execution controls:** sandbox modes, approval policies, network controls
- **Extension system:** skills, plugins, MCP, hooks, and subagents
- **Automation layer:** SDK, app-server protocol, GitHub Action, scheduled automations

## Implication
A shared control and extension model allows consistent workflows from solo use to enterprise-scale operations.
