# Agent Skills Design Principles

## Principles
- Keep each skill narrowly scoped and outcome-oriented.
- Make trigger conditions explicit (when to use / when not to use).
- Use concrete examples over abstract guidance.
- Prefer composability: many small skills over one mega-skill.

## Practical checklist
- Is purpose explicit in one sentence?
- Are prerequisites and constraints listed?
- Are edge cases and failure modes addressed?
- Is expected output format specified?
